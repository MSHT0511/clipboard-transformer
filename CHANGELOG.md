## [1.0.0] - 2026-05-03

### Added
- Initial release
- Clipboard text transformation on Ctrl+V
- Literal string replacement rules
- Regex pattern matching rules
- System tray integration
- Configuration file hot-reload
- Windows notification support
- Individual rule enable/disable
- PyInstaller executable build support
- Comprehensive test suite with pytest
- Code coverage reporting

### Features
- Real-time clipboard monitoring
- Global keyboard hook (requires admin privileges)
- JSON-based configuration
- Chain multiple transformation rules
- System tray icon with menu
- Logging to file

### Documentation
- README with installation and usage instructions
- EXE_USAGE for standalone executable
- Example configuration file
- Test documentation
